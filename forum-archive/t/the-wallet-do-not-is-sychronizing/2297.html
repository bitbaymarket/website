Craig_Claussen 2020-03-22 16:31:55 UTC #2

I am sorry to say that your blockchain data is corrupt. You will need to resync from the beginning.
In order to do this you need to first make a backup of your wallet.dat file

Then search for your …/appdata/roaming/bitbay folder. Make sure you have your windows allow you to ‘see hidden files’ setting on. Inside this folder you can delete everything but the wallet.dat file.

At this time you can use the bootstrap file and place it in this appdata/roaming/bitbay folder.
The bootstrap download is here: https://github.com/bitbaymarket/bitbay-bootstrap/releases/tag/v1

simply download the bootstrap.dat file and place in your appdata/roaming/bitbay folder.

You will only have 2 files in that folder.
wallet.dat
boostrap.dat

Then restart the QT wallet

Sorry about the inconvenience
