Pericle,

Sorry for the delay.
Understand that we had an issue with the blockchain splitting between the longest chain and the most trusted chain. We are almost done resolving the issue. Your coins are safe as long as you haven’t tried transferring them during the split.

You will most likely have to synchronize to the blockchain from the very beginning due to this chain split.

So the easiest way to fix your situation where you want to change from 32 bit windows to 64 bit windows is as follows:
1st) make sure you know the location of your wallet.dat file. Have it stored in multiple safe locations.

2nd) Locate your qt folder that stores all your data (not the .exe file). You can find the location of this folder by reviewing this video: https://www.youtube.com/watch?v=jGXf2xFyWaE

3rd) delete all content in this folder EXCEPT FOR your wallet.dat file. Also… delete your old 32bit qt wallet .exe

4th) download the bootstrap file found here: https://github.com/bitbaymarket/bitbay-bootstrap/releases/tag/v1
Place in the same folder as your wallet.dat file. This means you would only have 2 files in that folder. The wallet.dat and the bootstrap.dat file

5th)download and install the new QT wallet 64 bit version. Once extracted you can start the new .exe.
6th) let it synchronize

Let me know if you have any other questions
