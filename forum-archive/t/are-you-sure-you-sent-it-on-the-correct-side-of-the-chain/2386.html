Craig_Claussen 2020-04-21 09:58:06 UTC #2
pranilb:

v2.1.1.0-g32a928e

That is an old version of the QT wallet

First you need to back up your wallet.dat file.
Then you need to download the latest version from the website download page.
Then you need to clear every file except your wallet.dat file from your C:/username/appdata/roaming/bitbay folder
You can also download bootstrap file to speed up the synchronization process
If you decide to download it, you need to place the bootstrap.dat file into the same folder above.

That would mean you would only have 2 files in that folder: wallet.dat and bootstrap.dat
Then restart the QT wallet
