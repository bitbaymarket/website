Matej,

The Client does not use the wallet.dat to load up private keys.
Only the QT wallet uses wallet.dat to store private key.

When you create your account for the Client, it asks you to save your private keys to the computer. It actually forces you to save the keys to computer before it will allow you to create an account. So they should be on your computer somewhere.

Do a search for files with .private extension
