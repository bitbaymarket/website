How to update the Client and/or QT wallet to the latest version

***Note: All download links on the BitBay site point to the latest version

How to update the BitBay QT Wallet:

Find the download link for your platform here:

https://bitbay.market/downloads

  1. Backup your wallet before doing anything!
  2. Once the update has been downloaded and installed, simply replace the previous QT wallet file with the new one. That’s it! You are now running the latest version, which has Dynamic Peg voting capabilities.

How to update the BitBay Markets Client:

To reduce update issues for this peg-enabled version, we strongly recommend that Markets Client users perform a brand new install of the program, instead of running a basic update.

Find the download link for your platform here:

https://bitbay.market/downloads

  1. YOU MUST BACK UP BOTH PRIVATE KEYS in a SECURE location before starting.
  2. Uninstall/remove all files related to the existing markets client.
  3. Find the correct download for your OS at the link above, download it then install.
  4. Next, open the new BitBay program.
  5. For faster syncing , please see directions below for importing the new bootstrap file, with this bootstrap, syncing should only take a few hours.
  6. Once you are synced 100%, Load your two (2) private keys from the file menu (file->open->normal account) . For the first time, this must be done from the file menu only,. If you have multiple accounts load each one in any order.
  7. Once you finish loading the keys, it might show a zero balance until 100% synced.

For advanced users only

Using the Markets Client updater ( without doing a fresh install):

  1. BACK UP BOTH PRIVATE KEYS in a SECURE location before starting.
  2. Download the update here:

https://bitbay.market/downloads

  1. Run the updater, then close the program once finished.
  2. Rename “ wallet.dat” to “ wallet-original.dat” .
  3. Open the BitBay Markets Client.
  4. Once you are synced 100%, Load your two (2) private keys from the file menu (file->open->normal account) . For the first time, this must be done from the file menu only ; if you have multiple accounts load each one in any order.
  5. Once you finish loading the keys, it might show a zero balance. To refresh, simply click the rescan button on the lower right corner. “ Rescan ” is different from “ Re-sync ” as it just rebuilds wallet.dat only . Once that’s done you should be up to date, and your wallet balance will be current.

New (untested) Bootstrap Instructions:

If you are doing a fresh install for either QT Wallet or Markets Client, or do not have an existing version of the QT Wallet or Markets Client installed, here is a brand new bootstrap file which significantly speeds up the syncing process:

GitHub

bitbaymarket/bitbay-bootstrap

Contribute to bitbaymarket/bitbay-bootstrap development by creating an account on GitHub.

*** Note: This is bootstrap has not yet been heavily tested. If you experience any issues with it, please let the team know in Slack.

To use the bootstrap, first remove all files in the data folder (except wallet.dat, peers.dat, and any testnet files).

Then simply drop the bootstrap file above into the following directories after installing the program:

For Client users , the directory is:

PC Users: C:/BitBay/bitbaydata

Linux Users: ~/BitBay/bitbaydata

Mac Users: ~/Library > Application Support > BitBayClient > bbaydata

For QT users, the directory is:

Mac users: ~/Library > Application Support > BitBay

Linux users: ~/.bitbay (make sure you have “show hidden files” selected)

Windows users: ~/user/appdata/roaming/bitbay folder (make sure you have “show hidden files” selected)

Once the file has been added, restart the Client or Qt, and that’s it!

** If you have any questions at all or need help with the update, please let the BitBay team know in the “ #Support ” channel on Slack or Discord. *
