プログラム可能なマネーに革命を起こす

In 2014, the BitBay Halo Client was the industry's first to introduce a trustless, decentralized marketplace powered by unbreakable smart contracts.
Fast forward to the present, and we are making progress every day, bringing you the most feature-rich platform in crypto.

Completed 2024 - 2025

Web marketplace release

Community testing of the Solidity/Ethereum version of unbreakable contracts. Completion of the new GUI for web markets, lot's of new features, fully audited and built to scale. Affiliate system for users to self-promote. AI self-filtering of orders to keep market orders clean. Decentralized nodes that users can run for self-powering the system along with AI nodes, IPFS nodes, and rate limits based on actual users verifying requests.

BitBay Bridge Fork

Forking for the BitBay bridge completed with auditing of the solidity contracts. Planned release for the bridge and trading on popular AMM exchanges. Polygon, Solana, Binance and other networks are now accessible depending on which chains the community wishes to bridge to.

Growing the community

Users can now start to embrace revitalization and self marketing of the project.
Completed 2021 - 2023

BitBay code for bridge

Creation of solidity style mini-peg system with ability to bridge to any Ethereum style blockchain.

Staking curation system

Gives the community the ability to manage the side chains created by the bridge. Also gives them ability to more easily integrate into decentralized exchanges for both Solidity and BitBay chains.

AMM Solidity trading system

The BitBay solidity peg will be able to identify if users are interacting with AMM decentralized exchanges through a custom made router. Using some complex and innovative tactics, it makes buying and selling a seemless experience.

Web marketplace redesign for Halo

Unbreakable contracts released in testnet for Ethereum and Solidity chains. Fully decentralized web marketplace created for all approved ERC-20 tokens(such as DAI) and Ethereum for eventual release on popular chains such as Polygon. This expands the size of the BitBay community.

Integration into NightTrader decentralized exchange

BitBay trading went live on the decentralized exchange NightTrader. The security of the exchange uses threshold signatures to protect active trades and gives the user a key so funds on the exchange are in their control. After BitBays fork it will be able to take full advantage of the DEX security.
